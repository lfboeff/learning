	My List of MacOS Useful Commands (LFBoeff)
---------------------------------------------------------------------------------------------------------------


	Open file using specific application
---------------------------------------------------------------------------------------------------------------

$ open -a TextEdit filename

* -a flag specifies any application you want, including ones where TextEdit isn't the default editor.
* -t  opens in the default editor (i.e. if you use BBEdit, TextMate, etc.)
* -e will open the file specifically in TextEdit


	Close application (safe-mode)
---------------------------------------------------------------------------------------------------------------
$ osascript -e 'quit app "Slack"' >>> safe mode

$ pkill -x Slack >>> unsafe mode


	Check if git local repo is behind main branch
---------------------------------------------------------------------------------------------------------------
$ git remote update; git status -uno


	Check processor architecture
---------------------------------------------------------------------------------------------------------------
$ uname -m


	Log into mySQL installation using 'root' user ('rootuser') or 'golang'@'localhost' user ('golang')
---------------------------------------------------------------------------------------------------------------
$ cd /usr/local/mysql/bin/mysql -u root -p
